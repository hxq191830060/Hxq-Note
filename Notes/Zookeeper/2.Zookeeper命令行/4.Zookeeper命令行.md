* 查看节点下的子节点——ls [-s] [-w] [-R] path

  * ls path  

    返回path节点下的一级子节点集合

  * ls -s path

    返回path节点下的一级子节点集合+path节点的各个属性值

  * ls -R path

    返回path节点下的各级子节点集合



* 创建节点—— create [-s|-e|-es] path [content]
* 获取节点数据——get path
* 获取节点状态——stat path
* 修改节点数据——set path newContent
* 删除节点——delete path
